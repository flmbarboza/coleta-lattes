# Dashboard de Produtividade Docente

Sistema em Streamlit para coleta e análise automática
de currículos da Plataforma Lattes.

## Funcionalidades

- Download automático de currículos
- Extração de produção científica
- Indicadores CAPES-like
- Dashboard interativo

## Executar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
